# Condom

Protect your production app with catching undesirable input.

## Installation

Via Composer

``` bash
composer require obotoz/condom
```

## Usage

Publish configs with:

``` bash
php artisan vendor:publish --provider="Obotoz\Condom\CondomServiceProvider" --tag=config
```
